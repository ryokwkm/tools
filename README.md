
# filecat

**ファイルリストに基づき、複数のファイルの内容を、相対パスのヘッダー付きで一つに連結するツールです。** LLM用のプロンプト生成やコードレビューに役立ちます。

-----

## 主な機能

- 複数のファイルを一つに連結
- 相対パスのヘッダーを自動で追加
- 出力先を標準出力とファイルで切り替え可能
- 引数なしで実行時のデフォルト動作

-----

## セットアップ

1.  スクリプトを `filecat` という名前で保存します。
2.  ターミナルで実行権限を付与します。
    ```bash
    chmod +x filecat
    ```

-----

## 使い方

### 1\. 標準出力モード

入力ファイルを引数に指定すると、結果を\*\*画面（標準出力）\*\*に表示します。

- **コマンド**: `./filecat [入力ファイルパス]`
- **使用例**: `./filecat my_files.txt | pbcopy` (結果をクリップボードにコピー)

### 2\. ファイル出力モード

引数なしで実行すると、**デフォルトの入力ファイル**を読み込み、**デフォルトの出力ファイル**に結果を保存します。

- **コマンド**: `./filecat`
- **デフォルト設定**:
    - **入力**: `~/source/filecat/filepath.txt`
    - **出力**: `~/source/filecat/files.txt`

-----

## 設定

スクリプト内の設定を編集することで、動作をカスタマイズできます。

### 相対パスの起点

相対パスの起点としたいディレクトリ名をリストに追加します。

```bash
# filecat スクリプトの内部

search_dirs=(
  "my-project"
  "another-repo"
  "component-library"
)
```

### 入力ファイルの例

入力ファイルには、連結したいファイルの絶対パスを1行に1つずつ記述します。

```
/home/user/dev/my-project/src/main.js
/home/user/dev/another-repo/docs/api.md
/tmp/memo.txt
```
